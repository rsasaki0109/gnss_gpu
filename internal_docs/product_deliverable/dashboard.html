Interpretation cheatsheet

  • high tier (green): trust the route-level prediction directly; error expected ≤ 3 pp.
  • medium tier (orange): use with caution near decision boundaries; error 3-8 pp expected.
  • low tier (red): contains focus-case windows. Drill into the scatter / focus table before acting.
  • False-high markers (red dots on bottom-left): adopted model partially suppresses but residual inflation remains (Tokyo run2 w7/w9).
  • Hidden-high markers (purple on top-left): adopted model lifts but still undershoots (Tokyo run2 w23-w27).
  • False-lift markers (orange, upper-left): adopted model partially rejects (Nagoya run2) or still over-predicts (Tokyo run3 w17).
